Deletor

Powerful CLI utility for intelligent file management

Telegram

Features

Delete by Extension

Remove files with specific extensions like .mp4, .zip

Size Filter

Delete files larger than specified size (10mb, 1gb)

Recursive Search

Scan directories and all subdirectories

Confirmation Prompt

Safe deletion with confirmation

Table Output

Clean, formatted table display

Installation

go install github.com/pashkov256/deletor

Usage

deletor -e mp4,zip -d ~/Downloads/ -s 10mb

Arguments

-e, --extensions

List of file extensions separated by commas (e.g., mp4,zip,jpg)

-d, --directory

Path to the directory to search for files

-s, --size

Maximum file size (e.g., 10b, 10kb, 10mb, 10gb)

--exclude

Exclude specific files/paths (e.g., data,backup)

-p, --progress

Display a progress bar during file scanning

Examples

Basic Usage

deletor -e mp4,zip -d ~/Downloads/ -s 18kb
2.96 MB    /home/user/Downloads/sample.zip
155.14 KB  /home/user/Downloads/image.jpg
370.86 KB  /home/user/Downloads/document.png

7.48 MB  will be cleared.

Delete these files? [y/n]: y
✓ Deleted: 2.52 MB

With Progress Bar

deletor -e mp4,zip -d ~/Downloads/ -s 10mb -p

With Exclusions

deletor -e mp4,zip -d ~/Downloads/ -s 10mb --exclude backup,important
