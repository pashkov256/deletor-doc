Features

Interactive TUI

Modern text-based user interface for easy file navigation and management

Delete by Extension

Deletes files with specified extensions (e.g., .mp4, .zip)

Size Filter

Deletes only files larger than the specified size (e.g., 10mb, 1gb)

Directory Navigation

Easy navigation through directories with arrow keys

Quick Selection

Select and delete files with keyboard shortcuts

Rules System

Create and manage deletion presets for repeated use

Installation

go install github.com/pashkov256/deletor

Usage

TUI Mode (default)

deletor -d ~/Downloads/

CLI Mode (with filters)

deletor -cli -e mp4,zip -d ~/Downloads/ -s 10mb
